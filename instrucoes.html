Instruções para Uso do Site FBBRODONTO

Como Usar o Site Offline

1. Baixe o arquivo ZIP

Faça o download do arquivo fbbrodonto.zip através do link fornecido.

2. Extraia os arquivos

Extraia todos os arquivos do ZIP para uma pasta no seu computador ou pendrive.

Você pode fazer isso clicando com o botão direito no arquivo ZIP e selecionando "Extrair Tudo" (Windows) ou usando o aplicativo de descompactação do seu sistema.

3. Inicie o site

Abra o arquivo start.html em qualquer navegador moderno (Chrome, Firefox, Edge, Safari).

Este arquivo iniciará automaticamente o site FBBRODONTO, redirecionando para a página principal.

Dica: Se estiver usando um pendrive, basta clicar duas vezes no arquivo start.html para iniciar o site sem necessidade de instalação ou internet.

4. Navegue pelo site

Use o menu de navegação para acessar as diferentes seções:

  • Resumos - Conteúdo resumido sobre temas odontológicos
  • Questões - Questões de prova e concursos para praticar
  • Cursos - Cursos gratuitos em diversas áreas da odontologia
  • Assistente IA - Assistente virtual para tirar dúvidas

Requisitos do Sistema

O site FBBRODONTO foi desenvolvido para ser leve e funcionar em praticamente qualquer dispositivo moderno:

  • Qualquer navegador web atualizado (Chrome, Firefox, Edge, Safari)
  • Não requer conexão com internet
  • Funciona em Windows, Mac, Linux e dispositivos móveis
  • Espaço em disco: aproximadamente 5MB

Observação: O site foi otimizado para funcionar offline, mas algumas funcionalidades como o Assistente IA têm capacidades limitadas sem conexão à internet.

Suporte

Se você encontrar qualquer problema ao usar o site, verifique se:

  • Todos os arquivos foram extraídos corretamente
  • Você está usando um navegador web atualizado
  • Está abrindo o arquivo start.html e não outro arquivo
